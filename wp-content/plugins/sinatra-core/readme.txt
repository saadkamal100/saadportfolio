== Description ==

Sinatra Core is an optional companion plugin for [Sinatra](https://sinatrawp.com) theme. It adds additional features such as widgets, blocks and a collection of pre-built websites (demos) for popular page builders and the WordPress block editor (Gutenberg).

== Installation ==

= Automatic Installation =

Automatic installation is the easiest way to install Sinatra Core as WordPress handles the file transfers itself. To do an automatic install of Sinatra Core, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type "Sinatra Core" and click Search Plugins. Once youâ€™ve found our plugin simply click "Install Now". After the installation has finished, click the "Activate Plugin" button.

= Manual Installation =

1. Upload the "sinatra-core.zip" folder to your "/wp-content/plugins" directory or alternatively upload the "sinatra-core.zip" file via the "Add New" button in WordPress plugins page.
2. Activate Sinatra Core.

== Frequently Asked Questions ==

= I installed the plugin but nothing happened =

Sinatra Core is a companion plugin for the [Sinatra](https://sinatrawp.com) theme and will only work with the theme installed and activated.
